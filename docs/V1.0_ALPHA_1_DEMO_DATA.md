# V1.0-alpha.1 演示数据与快速启动

## 目标

让用户无需真实抓取、无需真实 LLM，也能快速体验完整主流程。

## 命令

### 创建演示数据

```bash
python scripts/create_demo_data.py
```

### 重置演示数据

```bash
python scripts/create_demo_data.py --reset-demo
```

### 启动服务

```bash
uvicorn app.main:app --reload --port 8779
```

## Demo 数据包含什么

创建脚本会在本地数据库中生成以下数据：

| 数据类型 | 说明 |
|---------|------|
| Source | source_key = `demo_ai_frontier` |
| SourceItem | 状态为 `compiled`，关联到演示卡片 |
| InsightCard | 状态为 `completed`，包含中文摘要、关键事实、技术洞察、产品机会、风险、行动建议 |
| InsightCardBilingualReport | 包含英文核心摘要、原文主张、证据点、术语对照、中文解说 |
| CardDecision | decision = `to_action`，备注说明用于演示 |

## 可以访问哪些页面

创建 demo 数据后，运行 `python scripts/create_demo_data.py` 会输出：

```
SourceItem:
  /source-items/{id}

InsightCard:
  /cards/{id}

Full report:
  /cards/{id}/export-report

Action task:
  /cards/{id}/export-markdown
```

这些页面共同构成完整演示链路：

1. `/source-items/{id}` — 查看演示资料
2. `/cards/{id}` — 查看 InsightCard，包含中英双语报告
3. `/cards/{id}/export-report` — 导出完整 Markdown 报告
4. `/cards/{id}/export-markdown` — 导出 Markdown 行动任务

## 首页演示入口

创建 demo 数据后，打开首页 `/`，右侧会出现 **🎬 演示数据入口** 区块，显示：

- 查看演示资料
- 查看演示 InsightCard
- 导出演示完整报告
- 导出演示行动任务

如果尚未创建 demo 数据，首页会提示：

```
还没有演示数据。可以运行以下命令创建：
python scripts/create_demo_data.py
```

## reset demo 数据

```bash
python scripts/create_demo_data.py --reset-demo
```

这会删除 `source_key = demo_ai_frontier` 对应的所有 demo 数据（Source、SourceItem、InsightCard、BilingualReport、CardDecision），然后重建。

## 边界

| 限制 | 说明 |
|------|------|
| 不影响真实数据 | 只操作 `source_key = demo_ai_frontier` 的数据 |
| 不联网 | 不调用任何外部 API |
| 不调用 LLM | 所有内容为预设的演示文本 |
| 不读取 .env | 不依赖任何环境配置 |
| 幂等 | 重复运行不会重复插入，会跳过已存在且完整的 demo 数据 |

## 验收脚本

验证 demo 数据可用于完整链路：

```bash
python scripts/acceptance_demo_data.py --isolated-db
python scripts/acceptance_demo_data.py --isolated-db --keep-db
```
