# V0.5 将 InsightCard 转成 Markdown 行动任务

## 目标

验证用户能把标记为"转成行动"的中文 InsightCard 导出为 Markdown 任务草稿。

## 产品闭环

```
英文资料
→ 中文 InsightCard
→ 标记"转成行动"
→ 导出 Markdown 任务
→ 复制给 AI 执行模型或保存到本地
```

---

## 前置条件

- 已安装依赖
- 数据库中有已完成编译的 InsightCard
- 不需要 LLM API Key

---

## 手动验收步骤

### 1. 准备一张"转成行动"的卡片

如果没有，可以用系统已有卡片，或通过验收脚本自动创建：

```bash
python scripts/acceptance_export_action_markdown.py --isolated-db
```

### 2. 打开 /cards?decision=to_action

确认列表中有"导出任务"链接。

### 3. 打开卡片详情页

点击"导出为 Markdown 任务"按钮（或"预览 Markdown 任务草稿"）。

### 4. 检查 Markdown 预览页

确认包含以下内容：

- 原文链接
- 中文摘要
- 关键事实
- 技术洞察
- 产品机会
- 风险与注意事项
- 行动建议
- 用户备注
- 可交给 AI 执行模型的任务草稿引导

### 5. 点击下载

确认下载文件名为 `insightcard-{id}-task.md`。

### 6. 检查非 to_action 卡片

在 /cards 列表中，非 to_action 的卡片不应显示"导出任务"按钮。

## 当前限制

- 只支持单张导出
- 不调用 LLM 二次改写
- 不自动创建任务系统
- 不同步 Notion / GitHub / 飞书
