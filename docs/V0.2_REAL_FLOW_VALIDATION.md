## 可选真实编译验收

前置条件：
- 已配置 LLM API Key（`MINIMAX_API_KEY`）
- 网络可访问
- SourceItem URL 可抓取

步骤：
1. 启动服务：`uvicorn app.main:app --reload`
2. 访问 `/source-items`
3. 进入某个 SourceItem 详情页
4. 点击"编译为 InsightCard"
5. 等待编译完成（同步请求，约 10–30 秒）
6. 查看 `/cards/{id}`

详情页操作区现在根据 `item.status` 显示不同提示：

|------|----------|

## 已知风险

- 真实网络不稳定，可能导致探测超时或失败
- 不同网站 HTML 结构变化可能影响 HTML Index 探测质量
- 手动编译为同步请求，耗时较长（10–30 秒），无中间状态提示
- 没有后台任务和 compiling 状态
- 没有生产级数据库迁移（Alembic）
- RSS feed 格式不规范可能导致部分条目解析失败
