# V0.8.2 真实 LLM 中英双语报告质量验收

## 目标

验证真实 LLM 生成的中英双语报告是否满足保真、语言边界和最低质量标准。

## 为什么要做

Mock 只能验证链路，不能验证模型是否真的遵守：
- 英文部分保留原文主旨
- 中文部分帮助理解
- 产品机会和行动建议不能冒充原文结论

---

## 验收命令

### Mock 链路验证（不调用真实 LLM）

```bash
python scripts/acceptance_bilingual_report.py --isolated-db --mock
```

### 真实 LLM 质量验收（需要 MINIMAX_API_KEY）

```bash
python scripts/acceptance_real_bilingual_report.py --isolated-db --real
```

### 保留 DB 方便排查

```bash
python scripts/acceptance_real_bilingual_report.py --isolated-db --real --keep-db
```

### 使用已有卡片

```bash
python scripts/acceptance_real_bilingual_report.py --card-id 1 --real
```

## 质量标准

V0.8.2 扩展了语言检查，确保英文字段为英文、中文字段为中文：

| 字段 | 语言 | 检查 |
|------|------|------|
| `english_core_summary` | 英文 | 非空 + 看起来像英文 |
| `english_key_claims` | 英文 | >= 2条 + 看起来像英文 |
| `english_evidence_points` | 英文 | 看起来像英文 |
| `chinese_explanation` | 中文 | 非空 + 看起来像中文 |
| `fidelity_notes_zh` | 中文 | 非空 + 看起来像中文 |
| `interpretation_boundary_zh` | 中文 | 非空 + 看起来像中文 |

### inspect_bilingual_report_quality() 返回字段（V0.8.2）

| 字段 | 类型 | 说明 |
|------|------|------|
| `english_summary_present` | bool | english_core_summary 非空 |
| `english_summary_looks_english` | bool | 英文字母显著多于中文字符 |
| `english_key_claims_count` | int | english_key_claims 数组长度 |
| `english_key_claims_look_english` | bool | 第一条主张看起来像英文 |
| `english_evidence_points_count` | int | english_evidence_points 数组长度 |
| `key_terms_count` | int | key_terms 数组长度 |
| `chinese_explanation_present` | bool | chinese_explanation 非空 |
| `chinese_explanation_looks_chinese` | bool | 中文字符 >= 4 且比例合理 |
| `fidelity_notes_present` | bool | fidelity_notes_zh 非空 |
| `fidelity_notes_look_chinese` | bool | 保真提示看起来像中文 |
| `interpretation_boundary_present` | bool | interpretation_boundary_zh 非空 |
| `interpretation_boundary_look_chinese` | bool | 解读边界看起来像中文 |
| `passed_minimum_quality` | bool | 所有字段通过语言+存在性检查 |
| `warnings` | list[str] | 质量警告列表 |

### 通过标准

```
english_core_summary 非空 AND 看起来像英文
english_key_claims >= 2 AND 看起来像英文
chinese_explanation 非空 AND 看起来像中文
fidelity_notes_zh 非空 AND 看起来像中文
interpretation_boundary_zh 非空 AND 看起来像中文
```

## 常见失败

| 失败类型 | 原因 | 解决方案 |
|----------|------|----------|
| API Key 缺失 | MINIMAX_API_KEY 未设置 | 设置环境变量 |
| LLM 返回非 JSON | 模型输出格式异常 | 检查 prompt 或模型配置 |
| 英文字段输出中文 | 模型混淆语言 | 检查 prompt 语言规则 |
| 中文字段输出英文 | 模型混淆语言 | 检查 prompt 语言规则 |
| 主张数量不足 | 模型未输出足够主张 | 检查 prompt 要求 |
| 保真提示为空 | 模型未输出 fidelity_notes | 检查 prompt 要求 |
