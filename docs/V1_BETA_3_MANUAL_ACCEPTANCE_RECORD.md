## 可接受限制

以下现象在 V1.0-beta.3 中经决策可接受：

| # | 限制 | 说明 |
|---|------|------|
| 1 | 最左侧深色全局导航仍是页面级跳转 | 不做全站无刷新 |
| 2 | 表单提交仍是传统 POST 请求 | 不做 SPA 表单 |
| 3 | JS 禁用时降级为完整页面跳转 | 渐进增强保证可用性 |
| 4 | 一次最多处理 5 条摘要 | `summary_limit=5` 是有意限制 |
| 5 | 当前是单用户本地 MVP | 不支持多用户权限 |
